# Progetto Web 2025-2026 - Centro Fitness

Repository per lo sviluppo del progetto di "Introduzione alla Programmazione per il Web".

## Struttura della Repo
* `/esempi`: Esempi di codice della profesoressa da cui partire.
* `/web_app`: Applicazione principale (Spring Boot, MVC, Frontend).
* `/RestLike`: Servizio REST-like per la gestione dei programmi di allenamento.
* `/relazione`: Documentazione e bozze per il file PDF finale.
